# research-productivity-diagnosis
A diagnostic compendium for the modern scholar, wherein the maladies of research—aimless wandering, unripe ideas, stalled manuscripts, and the cold silence of a blank page—are each met with their proper remedy.

# 简介

当用户描述科研困境、研究规划需求、论文写作瓶颈、或带着零散研究素材寻求方向时使用。覆盖四类场景：(A) 想做研究规划但方向未定；(B) 有想法但不确定是否值得做；(C) 论文卡住、写不下去或被审稿人质疑（含 motivation、method、reviewer 回应、故事线、摘要撰写、创新点凝练等子卡点）；(D) 只有零散关键词和已发表论文需要冷启动。触发信号包括："我想做 XX 方向"、"这个 idea 行不行"、"论文卡住了"、"reviewer 说……"、"我之前发过……现在想做……"、"帮我写摘要"、"创新点怎么提炼"等。主要适用范围：国自然 F02（计算机科学）、F03（自动化/信息科学）、A01（数学）领域研究者，即机器学习、NLP、CV、多模态、数据挖掘、控制、信息论、应用数学等方向。
